# Superhero store


Napravio sam Superhero store koja u suštini ima 4 marke tena i ispod neke modele "koji su trenutno na stanju".Koristio sam tabelu radi lakšeg sortiranja i estetike.Svaka od tena ima svoju cijenu i i opciju "Buy". U headeru je ime prodavnice i njen logo,dok se u footer nalaze basic informacije.Takoder na vrhu se nalazi i login forma kako bi korisnici imali svoje profile i mogli obavljati online kupovinu.
